Projet : Digitalisation des Documents pour la SNEL

Ce projet visait à créer une application de gestion électronique des documents (GED) pour la Société Nationale d'Électricité (SNEL) en RDC.

Description du Projet

L’application a été développée pour digitaliser et sécuriser les documents d’entreprise, simplifier leur accès et leur stockage, et améliorer la traçabilité. Elle utilise Vue.js pour l’interface utilisateur et Node.js pour le back-end.

Technologies Utilisées

  • JavaScript (Vue.js, Node.js)
  • Base de données NoSQL
  • Figma pour le design
  • Star UML pour la modélisation

Fonctionnalités Principales

  • Recherche de documents par agents, catégories, et autres critères
  • Ajout et suppression de documents
  • Interface rapide et conviviale
